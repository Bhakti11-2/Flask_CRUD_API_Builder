Flask CRUD API Builder

This utility helps you build boiler-plate code for sample REST API with simple CRUD functionality.

Object/Entity Name:



After deploying boilerplate.py, you will have below REST API:

  • List: GET /<entity>
  • Create: POST /<entity>
  • Retrieve: GET /<entity>/<id>
  • Update: PUT /<entity>/<id>
  • Delete: DELETE /<entity>/<id>
