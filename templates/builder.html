Flask CRUD API Builder

This utility helps you generate boiler-plate code for building REST API with simple CRUD functionality

Github

Key is API attribute name and value is data type (python)


After deploying boilerplate.py, you will have below REST API:
  • List: GET /<entity>
  • Create: POST /<entity>
  • Retrieve: GET /<entity>/<id>
  • Update: PUT /<entity>/<id>
  • Delete: DELETE /<entity>/<id>
